# AI-Powered Task Management System

## Project Overview
The AI-Powered Task Management System is a smart web application designed to help users organize and prioritize tasks efficiently.  
It uses AI-based logic to automatically assign task priorities based on deadlines and task urgency.

The system provides a clean and interactive interface to track productivity and visualize task progress.

---

## Features

- AI-based task prioritization
- Dynamic progress tracking
- Interactive dashboard
- Responsive and user-friendly UI
- Task creation and management
